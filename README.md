# 📝 Report Card Generator

## Description
A simple Java program that allows users to input student names and grades, then generates a report card with a letter grade and personalized feedback.

## Features
- Input student names and grades.
- Automatically calculates letter grades (A, B, C, D, F).
- Provides personalized feedback for each student.
- Displays all report cards when finished.

## How to Run
1. Clone or download the project.
2. Compile and run `Main.java`.
3. Enter student names and grades, then type `q` to quit and view the report cards.
